# Conversor de Moedas

Aplicativo para conversão de moedas desenvolvido utilizando Flutter e Dart e a API da HG Finance para consumo dos dados.

## Suporte:
- Real (R$)
- Dolar ($)
- Euro (€)
- Libra esterlina (£)
